# Images Folder

Put all your blog images here (screenshots, photos, diagrams, etc.)

## Usage

In your blog posts, reference images like this:

```markdown
![Description of image](/images/your-image.png)
```

## Tips

- Use descriptive filenames: `seo-agent-dashboard.png` not `screenshot1.png`
- Keep images under 1MB when possible
- Use PNG for screenshots (crisp text)
- Use JPG for photos (smaller file size)
- Optimize images before adding (use TinyPNG.com)

## Example

If you save an image as `public/images/my-screenshot.png`, reference it in your blog post as:

```markdown
![My awesome screenshot](/images/my-screenshot.png)
```
